# To-Do List Web Application

## Description
This is a simple To-Do List web application that allows users to add, view, and delete daily tasks.
It is built using HTML, CSS, and JavaScript and helps in managing daily activities efficiently.

## Features
- Add new tasks with title and description
- Display list of tasks
- Delete tasks from the list
- Simple and clean user interface

## Technologies Used
- HTML
- CSS
- JavaScript

## Learning Outcome
- DOM manipulation
- JavaScript event handling
- Basic data management using arrays or localStorage
